JavaVsNativeThreads is a simple app to test native pthreads vs Java code

Getting started: After checking out this project using subversion, 
use the IMPORT PROJECT 
in Android Studio file menu item to import this project. 

You may need to set the ndk.dir and sdk.dir directories
if you are working on a non-lab machine


Next steps: Use the Project view to find the jni c code.
Complete the missing code in the .c file
